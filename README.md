# customer_churn_-1-minor-
This report presents a machine learning–based customer churn analysis, detailing data preprocessing, exploratory insights, and model evaluation. Using logistic regression and random forest, it identifies key churn drivers and provides actionable strategies to improve retention and customer satisfaction.
Customer Churn Prediction
📌 Overview
This project focuses on predicting customer churn using machine learning techniques. By analyzing demographic, usage, and subscription data, the goal is to identify at-risk customers and provide actionable strategies to improve retention.
🎯 Objectives
- Analyze customer data
- Identify factors influencing churn
- Build predictive models
- Support businesses in retention strategies
📂 Dataset
- Demographics: Gender, age, region, household type
- Usage Patterns: Login frequency, feature utilization
- Subscription Details: Contract type, payment method, tenure
- Target Variable: Churn (Yes/No)
⚙️ Tools & Technologies
- Language: Python
- Libraries: Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
- Environment: Jupyter Notebook
🔧 Data Preprocessing
- Handle missing values (imputation/removal)
- Encode categorical variables (One-Hot/Label Encoding)
- Scale features (Standardization/Min-Max Normalization)
📊 Exploratory Data Analysis (EDA)
- Churn distribution visualization
- Feature relationships with churn
- Key insights:
- Higher churn in month-to-month contracts
- High charges increase churn probability
🤖 Models Used
- Logistic Regression (baseline, interpretable)
- Decision Tree
- Random Forest (best performance, robust against overfitting)
📈 Evaluation Metrics
- Accuracy, Precision, Recall, F1-score
- Confusion Matrix for trade-off analysis
- Best model: Random Forest
✅ Results
- Successfully predicts churn
- Identifies high-risk customers
- Provides actionable business insights
🚀 Business Impact
- Targeted Incentives: Loyalty discounts for high-risk customers
- Proactive Outreach: Automated check-ins for declining usage patterns
- Subscription Optimization: Adjust contract terms for high-churn demographics
